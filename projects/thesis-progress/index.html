毕设开题：面向强化学习智能体训练的关卡生成系统

毕设开题记录，记录研究背景、目标和技术方案
2025-12-01 · 1 min · Rika

研究背景#

强化学习智能体在固定环境中训练容易产生过拟合，导致泛化能力不足。

同时，游戏关卡依赖人工设计存在成本高、覆盖有限的难题。

研究目标#

设计并实现一套"程序化内容生成 + 强化学习"的集成系统：

  1. 自动生成多样化游戏关卡
  2. 在生成的关卡上训练智能体
  3. 提升智能体在未见关卡上的泛化能力

技术方案#

关卡生成#

  • 约束随机生成 + BFS可达性检测
  • 支持难度可控调节

环境构建#

  • 基于Gymnasium标准接口
  • GridWorld网格游戏环境

训练框架#

  • Stable-Baselines3
  • PPO算法

下一步计划#

  • 完成关卡生成模块
  • 构建Gymnasium兼容环境
  • 集成PPO训练
  • 设计对照实验
  • 强化学习
  • 程序化内容生成
  • Unity
  • Python
