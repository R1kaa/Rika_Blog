计算机视觉

基于YOLOv8的实时口罩检测系统

项目背景 在公共场所安全管理场景中，人工巡检效率低、成本高，难以实现实时、全覆盖的监控。本项目利用计算机视觉技术，实现对人员口罩佩戴情况的自动化检测，可应用于商场、学校、地铁站等人员密集场所的安全监管。 核心工作 1. 数据标注与管理 Label Studio标注平台 部署并配置Label Studio开源标注平台 设计口罩检测专用标注规范（人脸框 + 口罩佩戴状态） 完成数据集的标注、审核与管理流程 数据集构建 收集公共场所监控场景图片 标注正样本（戴口罩）和负样本（未戴口罩） 数据增强：随机裁剪、翻转、色彩变换等 2. YOLOv8模型训练 模型选型与训练 采用Ultralytics YOLOv8作为检测模型 预训练模型微调：迁移学习加速收敛 超参数调优：学习率、批量大小、IoU阈值等 性能优化 模型量化：INT8量化减少推理延迟 多尺度检测：提升小目标检测精度 NMS优化：改进非极大值抑制效率 3. Web服务部署 后端服务架构 FastAPI构建RESTful API服务 支持图片和视频流两种输入模式 异步处理：uvicorn + 多进程worker 实时推理 支持USB摄像头和RTSP视频流接入 目标追踪：SORT算法关联多帧检测结果 结果可视化：绘制检测框和置信度 项目成果 ✅ 掌握了深度学习项目完整流程：数据标注 → 模型训练 → 部署上线 ✅ 建立了对目标检测算法工程落地的系统认知 ✅ 模型在测试集上达到**92.3%**的mAP@0.5 ⏱️ 推理速度：45 FPS（RTX 3060 GPU） 💡 技术栈：Python · YOLOv8 · Label Studio · PyTorch · FastAPI · OpenCV · ONNX ...
